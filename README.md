# 📰 Fake News Detection

## 📌 Overview
This project detects whether a news article is **Real or Fake** using NLP and ML.

## 🚀 Features
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression model
- Streamlit web app

## 🧠 Model
- Algorithm: Logistic Regression
- Accuracy: ~90% (depends on dataset)
